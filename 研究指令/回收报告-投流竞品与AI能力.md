# 回收报告 · 投流竞品全景 + AI 能力边界

## 〇、一句话总判决（先看这个）
**平台自带的 AI 已经把"开计划 / 出价 / 选品 / 素材 / 全托管"这条技术链吃掉了一大半，而且免费、数据最全、正在变成"超级经营智能体"。** 一个新做"投流 AI"的项目，在**单平台算法层几乎没有空间**；真正还空着的位置是三层：① 跨平台的战略协调层、② 干净归因 / 增量验证层（平台自报数据有利益冲突）、③ "人（专家）+ 工具"的服务层。这与红队 6-04"扛巨头自建引力"的判断完全一致——巨头不是将来会自建，是**现在进行时**。

## 二、Block B · 服务店家的第三方竞品

### 国外：已经分成清晰的三层 + 创意层
- **第 1 层 · 平台原生（免费）**：PMax / Advantage+ —— 平台内自动优化，第三方很难再增量。
- **第 2 层 · 平台专精优化**：Madgicx（Meta，自主出价/受众/创意分析，约 $49–79/月）、Optmyzr（Google）、Revealbot（跨平台规则引擎）；Amazon 侧 Perpetua / Pacvue / Teikametrics。
- **第 3 层 · 跨平台 AI agent 操盘（新兴、增长最快）**：Synter（"你定方向、agent 执行"，14 个广告平台 API 直连、160+ agent 工具）、Albert.ai（高自主，适合 >$10万/月预算）、Hyper、Percuity、Ryze AI；并出现把 agent 直连 Meta Ads API 的 **MCP server**（Pipeboard、GoMarble、官方 Meta MCP）。
- **创意层**：AdCreative.ai、Pencil、Smartly.io（创意自动化）、Predis、Quickads。
- 行业底数：IAB Europe 2025 报告称约 81% 公司已使用 AI 营销工具（来源：designrush，2025–2026）。

> 【推断·对 ZJPT 重要】这些成熟轮子**几乎全是西方平台生态**。京东/抖音/阿里是相对封闭的生态，第三方 API"接管投放"的能力弱，国外轮子大多接不进京东。→ **朋友的京东礼品店，可整合的"成熟轮子"主要是平台自家工具（京准通智能托管/京小通），不是这些海外 SaaS。**

### 国内：第三方主力是"代运营 + 平台自带"，纯第三方投放 SaaS 偏弱
- 代运营（TP）六家上市公司 2025 业绩分化严重：宝尊 2025 上半年仍亏约 9704 万、丽人丽妆上半年净亏约 3300 万（同比骤降超 1300%）；行业普遍因**流量碎片化 + 获客成本攀升 + 为品牌买流量的利润空间被挤压**而承压，多家明确转型"品牌管理/自营"（来源：21经济网/新浪财经/网经社，2025）。
- 例外：若羽臣靠自有品牌逆势高增（2025 Q3 营收同比 +123%），说明**靠"买量服务费"活不下去，靠"自己变成品牌"才活**——对"做服务方"是警示。
- 国内独立投放 SaaS（光云/店小秘/聚水潭等）具体投放 AI 能力：【本轮未深挖，需补】。

## 三、Block C · AI 目前能做什么 / 做不到

| 能力维度 | 成熟度 | 证据 |
|---|---|---|
| 素材生成（图/文/视频 AIGC） | ✅成熟 | 千川采纳率约 80%；AdCreative 评分；阿里"万相智造"秒级生成 |
| 出价 / 预算分配 | ✅成熟 | 平台自动出价；Madgicx 自主重分配预算且不触发学习期重置 |
| 选品 / 定向 / 关键词 | ✅较成熟 | AI 选品、无关键词定向、一键 lookalike |
| 计划搭建 / 全流程托管 | ✅成熟 | 全域推广 / PMax / AI 小万"一句话托管" |
| 数据分析 / 诊断 | ✅成熟 | "AI Analyst"自然语言洞察 |
| 跨平台统一 + agent 操盘 | 🟡新兴 | Synter/Albert 跨 14 平台，尚需大预算与信任 |
| **干净归因** | ❌做不好 | 平台用自家数据判自家广告＝利益冲突；小广告主转化量不足做不了增量实验（见第四节） |
| 长期是否跑赢人工 | ❓存疑 | 640 次增量测试显示 Advantage+ 长期未必赢人工 |

**一句话**：AI 已经能"把投放跑起来并跑得不差"，但**不能告诉你它到底有没有真带来增量**——这正是 ZJPT 月一地基要啃的硬骨头。

## 四、归因方法借鉴（直接喂养朋友店月一作战图）
- **分清两件事**：归因（attribution）是给已发生的成交分功劳；增量（incrementality）是用对照实验估"因为投了才多出来的"。两者不是一回事。
- **金标准**：geo holdout / conversion lift —— 把受众或地域随机分成投放组与对照组，比差值即增量（来源：Triple Whale/Haus/Stella，2025–2026）。
- **次选**：on-off 脉冲（简单但被外部因素/时间混淆，适合快反馈渠道）；MMM/历史建模（不用停投但弱）。
- **⚠️ 对单店礼品店的硬约束**：增量实验需要足够转化量与统计功效（常需 holdout 20–50% 受众、跑数周至数月）；**小广告主 / 低转化利基品类往往达不到**（来源：Haus，2025）。→ 朋友店若转化量小，严格 geo/lift 可能跑不出显著性。
- **结论回喂作战图**：月一"同店内并行 A/B"是务实替代，但要正视**样本量天花板**——可能需拉长周期、扩 SKU 组、或后续多店并样；且**不能只信平台后台 ROI**（自报有利益冲突）。

## 五、效果付费 / 结果分成在投放领域能不能活
- CPS / 纯效果付费把风险转给媒体/服务商，**媒体普遍不接**：因为供给方要替广告主的产品、运营、品牌全部风险兜底，又无法实时监控所有细节（来源：界面/知乎，行业共识）。
- 能活的窄缝：单价够高 + 分成比例够高 + 有完整交易闭环；且起量慢（约 3–6 个月）。
- 【推断·呼应红队】这正是 ZJPT"结果共担"的市场印证：**结果付费让供给方扛不可控风险**，与红队"对赌尸体"和"风险代理律师 capped survivor"一条线——结果客观可判 + 执行在供方手里，才可能成立。所以 ZJPT 的地基仍是"投放结果可控 + 可干净归因"，地基不成立就会重蹈"CPS 媒体不接"的覆辙。

## 六、对 ZJPT 的五条回写答案（最重要）
1. **差异化空间**：单平台算法层（出价/选品/素材/托管）已被平台原生 AI 吃掉且免费。空间只剩：跨平台战略协调层、干净归因/增量验证层、"专家+工具"的服务层。**纯做"更好的投流算法"没戏。**
2. **可整合轮子**：海外有成熟廉价的轮子（Madgicx/Optmyzr/Revealbot/AdCreative/Synter+MCP，$49–79/月起），但都绑西方平台，**接不进京东**。京东礼品店能整合的主要是平台自家工具（京准通智能托管/京小通）。自研投流 AI1 在京东场景的技术增量空间小。
3. **巨头引力威胁（6-04）**：极高，且已是进行时。京小通 / AI 小万 / 千川全域 = 巨头自建"超级经营智能体"吃掉第三方优化层的活体。**这是 ZJPT 投流 AI 地基的最大外部威胁。**
4. **归因方法**：金标准是 geo holdout / conversion lift；但单店低转化大概率达不到统计功效——这是月一作战图必须正视的天花板，不是细节。
5. **效果付费活法**：投放领域纯效果付费媒体普遍不接；ZJPT 要做结果共担，必须先过"结果可控 + 可干净归因"这道地基，否则模式从根上不成立。
