# 深研指令 · 投流竞品全景 + AI 能力现状

## 0. 一句话目的
摸清「帮店家做投流」这件事，**平台自己（自带AI）做到了哪一步、第三方玩家（国内外）在做什么、靠什么活、AI 目前真能解决哪些问题**——从而回答 ZJPT 投流AI引擎的唯一战略问题：**在平台已经把投放 AI 化的今天，"专家 + 我们的投流AI" 还有没有增量缝？**

---

## 1. 必答的两个核心战略问题（最锋利的，放最前；其余都是为这两个服务）

**Q1（地基威胁）**：平台自带的投放 AI——京东智能投放/京准通、巨量千川、万相台无界、Meta Advantage+、Google Performance Max、Amazon 自动投放——**已经把出价、预算分配、人群定向、创意生成自动化到哪一步了？** 在平台都"一键托管"的前提下，一个第三方"投流AI + 人"还能比平台自动投放多做什么、多赚什么？把平台**留给第三方/人的缝**逐条列出来；如果缝很窄，直接说。

**Q2（活法）**：真正活下来、赚到钱的"服务店家投流"玩家，**靠什么收费活久了**——工具订阅 / 按投放额抽佣 / 代运营服务费 / 结果分成？哪种模式存活率高、哪种是死亡模式（对照 ZJPT 已知死法：撮合佣金=猪八戒、结果对赌=锤子/斯微、大客户去中介化=宝尊/丽人丽妆）？

---

## 2. 研究范围（五块，每块都要给**真实玩家名 + 硬数据**，不要泛泛而谈）

### 块 A · 投放系统（平台侧 / 商家在哪投）
- 国内：京东（京准通 / 快车 / 海投 / 智能投放）、淘系（万相台无界 / 直通车 / 引力魔方）、拼多多（多多搜索 / 场景）、抖音（巨量千川）、快手（磁力金牛）、小红书（聚光 / 蒲公英）。
- 国外：Amazon Ads、Meta Advantage+、Google Performance Max、TikTok Ads、Temu / Shopee 商家投放。
- 每个要：① 自动化/AI 程度（出价/定向/创意各自动到哪步）；② 商家手里还剩哪些可控杠杆；③ **京东礼品 / 中小成长店实际能用到、用得起的是什么**。

### 块 B · 服务店家投流的竞品（国内）
代运营/TP（含已知死法宝尊、丽人丽妆）、专做投放的代理、第三方投放 SaaS / 自动化工具、AI 投放新创。
要：玩家清单 · 定位 · **收费模式** · 规模/融资/存活状况 · 面向中小店还是大店 · 是否覆盖京东类目。

### 块 C · 服务店家投流的竞品（国外）
Perpetua、Pacvue、Teikametrics、Helium 10、Skai（原 Kenshoo）、Smartly.io、Albert.ai 等。
要：做什么 · AI 到哪步 · 收费模式 · 客户层级 · **可迁移到中国/京东的程度**（哪些能搬、哪些水土不服）。

### 块 D · AI 目前在投流上真能做什么（2026 现状，逐项判成熟度）
自动出价/预算分配 · 人群与关键词 · 否词/搜索词优化 · 创意生成（图 / 短视频 / 标题文案） · ROI 预测与异常诊断 · 自然语言 agent 操盘。
每项标：**【已成熟】/【半成熟】/【还不行】**，并标明**平台自带已经做了 vs 留给第三方/人的缝**。

### 块 E · 归因方法与"专家增量"（直接喂红队地基）
- 行业怎么解投放 ROI 的**增量归因**（增量测量 / A-B / holdout / 时段交替 / MMM）？有没有现成方法论能直接借给朋友店月一作战图？
- "人（专家）+ AI" 相比"纯平台自动投放"，业界有没有**增量证据**（谁做对照实验、结果如何）？

## 3. 回收报告输出结构（报告该长这样）
1. 结论先行：对 Q1、Q2 的直接回答。
2. **竞品对照表**：名称 | 国别 | 定位 | AI 能力到哪步 | 收费模式 | 客户层级 | 存活状况 | 对 ZJPT 的启示。
3. **AI 能力成熟度表**（块 D 逐项 + 平台已做/留缝）。
4. 对 ZJPT 的 3 条战略含义（尤其："专家+AI"的增量缝到底在不在、有多宽）。
5. 仍空的坑（查不到、需下田验的）。

## 4. 信源与纪律（硬约束）
- **时效**：优先 2024–2026；投放产品迭代快，务必标产品版本与日期。
- **中国优先**：主语是京东礼品 / 中小成长店；国外只取可迁移的部分。
- **数据>推演**：要真实玩家名、真实收费、真实规模/融资/案例；区分【已核实】/【推测】；查不到就写"未找到"，**不许编**。
- **信源优先级**：平台官方文档/产品页、招股书/财报、一手报告 > 行业媒体 > SEO 软文（少用）。
- **边界复述**：本研究是周边情报，不验承重墙、不得判 Go/Kill；地基仍靠朋友店月一作战图。
